# 潜水培训平台 - 设计规范

## 设计理念
参考 Coursera、Udemy、edX、Khan Academy 等顶级在线教育平台
- **简洁**：减少视觉噪音，突出核心内容
- **现代**：使用现代化UI组件和交互
- **清晰**：信息层级分明，操作直观
- **一致**：全站统一的视觉语言

## 色彩系统
```
主色: #2563eb (蓝色 - 信任、专业)
辅色: #10b981 (绿色 - 成功、进度)
警告: #f59e0b (橙色 - 提醒)
危险: #ef4444 (红色 - 错误)
中性: #64748b (灰色 - 文字、边框)
背景: #f8fafc (浅灰 - 背景)
白色: #ffffff (卡片背景)
```

## 布局规范
- 页面最大宽度: 1280px
- 侧边栏宽度: 240px (收起: 64px)
- 卡片圆角: 12px
- 内容边距: 24px
- 组件间距: 16px

## 字体规范
- 标题: 24px / 20px / 16px
- 正文: 14px
- 辅助: 12px
- 行高: 1.6

## 组件规范
- 按钮: 圆角 8px，高度 40px
- 输入框: 圆角 8px，高度 40px
- 卡片: 白色背景，阴影 sm
- 标签: 圆角 6px

## 页面结构
```
┌─────────────────────────────────────┐
│  Header (固定顶部)                    │
│  Logo + 导航 + 用户信息               │
├──────┬──────────────────────────────┤
│      │                              │
│ Side │   Main Content               │
│ Bar  │                              │
│      │                              │
└──────┴──────────────────────────────┘
```

## 核心页面
1. 登录页 - 简洁、聚焦
2. 仪表盘 - 数据概览、快捷入口
3. 课程学习 - 沉浸式阅读体验
4. 进度中心 - 可视化学习进度
5. 管理后台 - 清晰的数据展示
